Значения, используемые в выражении - операнды

| Операция | Действие                                                                |
| -------- | ----------------------------------------------------------------------- |
| +        | Сложение                                                                |
| -        | Вычитание                                                               |
| \*       | Умножение                                                               |
| \\       | Деление. Если операнды - целые числа, результат - доля частного         |
| %        | Остаток от деления. Выполняется по следующему правилу: (a/b)\*b + a%b=a |
